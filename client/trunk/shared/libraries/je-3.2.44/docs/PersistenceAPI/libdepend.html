Library Dependencies

To use the DPL:

  • You must use Java 1.5.

  • Use the Berkeley DB, Java Edition version 3.0 jar file, just as if you were using the Berkeley DB, Java Edition APIs themselves (see the Getting Started with Berkeley DB Java Edition guide for more information). This jar file includes all the APIs that you need in order to use the DPL.
